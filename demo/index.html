Transparent/Humanized Message Pattern

Thanks to: Aza Raskin @ humanized and to Michael Heilemann for the jQuery implementation that prompted this (and whom designed the demo).

This is a prototype implementation of the same concept. jQuery's got the basic effects and addBehavior stuff baked in, so those are available but optional here (via scriptaculous and lowpro, naturally). One thing I did was seperate out the message logging functionality entirely. Do whatever you need to do with regards to a logger -- so long as it responds to a write(msg) method we're in business.
